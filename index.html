// How it works

01

Open the Steam Console

gdrevert fires steam://open/console and gives you the exact depot command to paste — no memorizing manifest IDs.

02

Run the Depot Download

Paste the command into Steam's console. Steam downloads the v2.2074 depot to steamapps/content/app_322170/depot_322171.

03

Check Readiness

The tool validates your Steam path via the Windows registry, confirms the depot files exist, and checks that GD isn't running before touching anything.

04

Backup + Sync

Your current GeometryDash.exe and libcocos2d.dll are saved to a timestamped backup folder. Then all depot files are copied over.

05

Lock the Manifest

appmanifest_322170.acf is set to read-only. This is the step most people miss — it stops Steam from silently updating GD back to 2.208+ on next launch.

06

Geode Check

gdrevert scans for Geode.dll or the geode/ folder. If Geode isn't installed, the installer page opens automatically.

◈ Why manifest 7678373534998244044?

This manifest ID maps precisely to GD v2.2074 — the last version with full Geode mod ecosystem support. Using the ID instead of just a version number guarantees you get the exact build every time, regardless of what Steam has in its local cache. The ACF lock on appmanifest_322170.acf makes sure it stays that way until you're ready to update.

// Features

🗂

Registry Detection

Finds your Steam install via the Windows registry. Works on C:, D:, E: — any drive.

💾

Timestamped Backups

Saves your existing EXE and DLL before overwriting, every time.

🔒

ACF Manifest Lock

Sets the manifest read-only so Steam can't silently roll you forward.

🔓

One-Click Revert

Unlock the manifest whenever you're ready to update GD back to latest.

⚙

Geode Detection

Checks for Geode post-downgrade and links you to the installer if it's missing.

📋

Live Log + Progress

Inline progress bar and file-by-file log so you know exactly what's happening.

🛑

GD Running Check

Blocks the downgrade if Geometry Dash is open — no corrupt files.

📦

Zero Dependencies

Pure Python stdlib + Tkinter. No pip installs. Just run the script.

// Requirements
